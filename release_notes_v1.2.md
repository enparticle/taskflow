## 🗂 네비게이션 & 구조 개선

### 사이드바 정리
- 칸반 보드 비활성화 (코드 유지, 필요 시 복구 가능)
- 내 업무 → 업무로 통합 (전체/내 업무 토글)
- 캘린더 메뉴 추가
- 업무 트리 메뉴 추가
- 전체 현황 보기(📺) 추가 — Admin/Viewer만 표시

### 프로젝트 상세 탭 개편
- 기존 6개 탭 → 3개로 축소 (개요 / 업무 / 팀)
- **계획 탭 제거** — 단계(마일스톤)를 업무 탭 안으로 통합
- 업무 탭에서 단계별 그룹핑, 접기/펼치기 지원
- 단계 관리 전용 모달 분리 (프로젝트 수정과 분리)

## 📅 캘린더

### 신규 캘린더 페이지
- 주별(기본) / 월별 뷰 전환
- 개인 일정 직접 입력 — 유형(개인/연차/휴일/미팅/마감), 색상, 시간, 팀 공개 여부
- 업무 마감일 자동 표시
- 날짜 클릭 시 새 일정 추가
- **역할별 개인화**
  - 일반 사용자: 내 일정 + 팀 공개 일정 + 내 담당 업무 마감일
  - Viewer: 전체 공개 일정 + 전체 업무 마감일

## 📺 전체 현황 뷰어 페이지 (`/viewer`)

- Admin/Viewer 전용 슬라이드 대시보드
- **슬라이드 구성**: 팀 전체 현황 → 프로젝트별 상세 → 4주 캘린더
- 자동 전환 (15초/30초/60초 선택)
- 전체화면 지원, 마우스 호버 시 일시정지
- 하단 진행 바, 슬라이드 인디케이터
- 캘린더: 지난 주 / 이번 주 / 다음 주 / 2주 후 표시

## ✅ 업무 관리 개선

### 필터/정렬 기능
- 검색, 상태 필터, 우선순위 필터, 정렬 (상태/우선순위/마감일/등록일/이름순)
- 완료 보기/숨기기 토글

### 일괄 작업 (Bulk Action)
- 체크박스로 여러 업무 선택
- 단계 변경 / 프로젝트 변경 / 상태 변경 / 우선순위 변경 일괄 적용

### 업무 상세 패널 개선
- 섹션별 접기/펼치기 (설명/세부 정보/댓글은 기본 펼침, 타임라인/변경이력/의존성은 기본 접힘)
- 최종 수정자 표시 (제목 아래)
- 변경 이력에 수정자 이름 표시
- 담당자 고정 색상 (사용자 ID 해시 기반)
- 상태 변경 즉시 반영 (낙관적 업데이트)
- 설명/제목 편집 state 분리 (동시 편집 시 내용 유실 버그 수정)

### 권한 체계 강화
- Admin/Leader만 업무 수정 가능 (제목, 설명, 담당자, 마감일, 우선순위 등)
- 상태 변경은 Admin/Leader + 담당자
- 프로젝트 내 새 업무 추가, 수정, 단계 관리 — Admin/Leader만
- 대시보드: Leader는 내가 속한 프로젝트만 표시

### 기타
- 댓글 수 업무 카드에 표시 (💬 N)
- 댓글 수정 기능 추가 (본인 댓글만, 수정됨 표시)
- 마감일 변경 시 담당자에게 알림 발송
- 업무 탭 기본값: 내 업무

## 📝 회의록 개선

### 음성 인식
- 클라이언트에서 직접 OpenAI Whisper API 호출 (Vercel 용량 제한 우회)
- 10분 단위 자동 분할 녹음 → 각 조각 즉시 변환 후 합산 (시간 제한 없음)
- 녹음 완료 시 자동 다운로드 (백업)

### 업무 분배
- 분석 결과에서 업무별 프로젝트 개별 선택 가능
- 프로젝트 지정 업무 → `task_drafts`로 저장 (리더 승인 대기)
- 프로젝트 미지정 업무 → 바로 등록

### 리더 검토 패널 (TaskDraftPanel)
- 프로젝트 개요/업무 탭 상단에 표시 (Admin/Leader만)
- 승인 / 수정 후 승인 / 반려 기능
- 수정 모드: 제목, 우선순위, 담당자, 마감일 변경 가능

### 회의록 묶기
- 10분 분할 녹음으로 생긴 여러 회의록을 하나로 묶기
- 묶음 제목 지정, 이름 변경, 그룹 해제 가능

### DB 추가
```sql
alter table meeting_drafts add column group_id uuid;
alter table meeting_drafts add column group_title text;
alter table meeting_drafts add column group_order int;
create table task_drafts (...);
```

## 🔧 버그 수정

| 항목 | 내용 |
|------|------|
| 비밀번호 재설정 | Supabase redirect URL 수정 (localhost → 배포 URL) |
| 상태 변경 즉시 반영 | 낙관적 업데이트 적용 |
| 댓글 수정 오류 | updated_at 컬럼 추가 후 정상 작동 |
| 사이드바 인코딩 | 한글 깨짐 전면 수정 |
| 업무 페이지 목록 미표시 | myOnly/myUserId state 선언 누락 수정 |
| 설명 저장 후 제목 수정 시 내용 유실 | 편집 state 분리 |
| 회의록 분석 실패 (Vercel 4.5MB 제한) | 클라이언트 직접 호출로 전환 |
